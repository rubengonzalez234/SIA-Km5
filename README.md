# SIA-Km5 - Sistema de Gestión Comunitaria

## 📋 Descripción
Sistema de gestión para comunidades con funcionalidades de:
- Gestión de miembros
- Control de finanzas
- Generación de reportes PDF
- Marca de agua en documentos

## 🚀 Tecnologías
- **Backend**: Django 5.1.4
- **Base de datos**: MySQL
- **PDF**: PyMuPDF, xhtml2pdf, reportlab
- **Imágenes**: Pillow

## 📦 Instalación Local

### Requisitos
- Python 3.8+
- MySQL 8.0+ o MariaDB 10.5+
